Projects

Patient Monitoring Robot

Robot designed to monitor the health of assisted living patients. Follows the patient and detects abnormal motions (such as falling) utilizing an Intel® RealSense™ camera alongside object recognition algorithms (YOLO, MediaPipe). Written in Python and implemented using ROS2 Foxy. I was responsible for implementing patient tracking as well as ROS2 integration and testing. Team Capstone Project sponsored by Intel. Awarded the Pioneering Concept Industry Choice Award at Oregon State University’s 2023 Engineering Expo .

Project Repository

AMD/Radeon LOD Tweaker

Program that locates and writes to Windows registry values automatically in order to modify level-of-detail (LOD) in DirectX9.0 and lower games. Only works on AMD GPUs.

Written in C#

Project Repository

OpenGL Keyboard Visualization

A 3D visualization of a keyboard that visually responds to real-life keypresses. Shading is included for toggling backlight options.

Written in C++ and OpenGL Shading Language (GLSL)
