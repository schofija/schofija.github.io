Projects

Patient Monitoring Robot

Team Capstone Project sponsored by Intel. Robot designed to monitor the health of assisted living patients. Follows the patient and detects abnormal motions (such as falling) utilizing an Intel® RealSense™ camera alongside object recognition algorithms (YOLO, MediaPipe). Written in Python and implemented using ROS2 Foxy. I was responsible for implementing patient detection/tracking, patient following, and ROS2 integration and testing of the fall detection algorithms.

⭐ Awarded the Pioneering Concept Industry Choice Award at Oregon State University’s 2023 Engineering Expo.

AMD/Radeon LOD Tweaker

Program that automatically locates and writes to Windows registry values automatically in order to forcefully modify level-of-detail (LOD) in DirectX9.0 and lower games. Due to differing registry structures between vendors, this program only works on Radeon GPUs.

Written in C#.

🔗 Github Repository
